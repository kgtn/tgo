## 🚀 为什么选择 TGO Web？

在 AI 浪潮下，客服不再只是简单的对话框。TGO Web 紧跟技术前沿，将 **React 19**、**SSE 流式渲染** 与 **MCP (Model Context Protocol)** 深度融合，打造出了一个既强大又丝滑的开发者利器。

### 🎨 技术栈：站在巨人的肩膀上
TGO Web 不仅仅是一个项目，它是现代前端技术的集大成者：
- **React 19 & Router 7**: 拥抱最新生态，享受极致的并发渲染性能。
- **TypeScript 5.9**: 全量类型安全，让每一行代码都有迹可循。
- **Vite 7**: 秒级启动，极致的开发调试体验。
- **Tailwind CSS 4**: 原子化样式的巅峰，构建现代 UI 从未如此简单。
- **Zustand**: 轻量级状态管理，告别冗余的样板代码。

## ✨ 核心亮点深度剖析

### 1. 🤖 真正的“智能”Agent 系统
不同于市面上简单的问答接口，TGO Web 支持 **多 AI Agent 灵活配置**。
- **SSE (Server-Sent Events) 流式响应**: 告别等待，AI 回复实时呈现。
- **上下文记忆**: 像真人一样沟通，AI 能够完美理解对话历史。
- **多模型支持**: OpenAI, Anthropic, DeepSeek... 随心切换。

### 2. 💬 悟空 IM (WuKongIM) 深度集成
在通讯底层，我们选择了 **WuKongIM**。
- **WebSocket 长连接**: 极低延迟，消息送达即刻确认。
- **状态同步**: 已读、未读、送达状态一目了然。
- **多媒体支持**: 文本、图片、文件，全方位满足沟通需求。

### 3. 🧩 革命性的 UI Widget 系统
这是 TGO Web 最具特色的功能。当 AI 返回结构化数据（如订单、物流、商品）时，TGO Web 会自动将其解析为精美的 **交互式卡片**。

#### 开发者友好的 Action URI 协议
我们定义了一套标准化的交互协议，只需简单的 URI 即可触发复杂操作：
- `url://`：外部链接跳转
- `msg://`：自动发送快捷指令到聊天
- `copy://`：一键复制单号、地址等

### 4. 🔧 拥抱未来：MCP 工具集成
TGO Web 深度集成了 **Model Context Protocol (MCP)**。
- **工具商店**: 丰富的 MCP 工具库，一键开启 AI 的“手脚”。
- **OpenAPI 自动解析**: 只需提供 Schema，AI 就能自动生成交互表单。

### 5. 🌐 全渠道接入：打通企业私域流量
TGO Web 不仅仅局限于 Web 端，它是一个全渠道的 AI 交互中枢。我们已经原生支持：
- **企业微信机器人**: 深度适配企微生态，支持自建应用接入，让 AI 员工直接进入企业通讯录。
- **微信公众号**: 轻松实现公众号菜单与对话框的 AI 化升级。
- **全平台支持**: 无论是 H5、移动 App 还是小程序，一套逻辑，到处运行。
